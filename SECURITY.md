# Security Policy for Echosync

## Supported Versions

Echosync is actively maintained. Security updates and patches are provided for the current major stable release. 

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| < 2.0.0 | :x:                |

*(Note: Versions prior to 2.0.0 were pre-stable and are no longer supported or patched.)*

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests. 

Instead, please report them using **GitHub's Private Vulnerability Reporting**:
1. Go to the **Security** tab of this repository.
2. Click on **Advisories** on the left sidebar.
3. Click the **Report a vulnerability** button.
4. Provide a detailed description of the issue, the steps to reproduce it, and any potential impact.

### What to expect
* We will acknowledge receipt of your vulnerability report within 48 hours.
* We will triage the issue and determine if it is a valid vulnerability.
* If accepted, we will work on a patch and keep you updated on the progress.
* Once the patch is ready, we will publish a security advisory and credit you for the discovery (if you choose to be credited).
